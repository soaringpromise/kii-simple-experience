A lightweight mod that simplifies and improves Minecraft's experience, enchanting, and anvil mechanics while staying close to vanilla gameplay.

The goal is to reduce unnecessary grind and randomness while keeping the system predictable, configurable, and balanced. Most features can be individually enabled, disabled, or adjusted through the configuration file.

---
## 📦 Features

<details>
<summary>📈 <b>Experience Progression</b></summary>

- **Linear XP progression:** Replaces the default exponential level curve with a fixed XP requirement per level. *(Toggleable)*
  - **Configurable XP per level:** Define exactly how much experience each level requires.

</details>

<details>
<summary>🪦 <b>Death Mechanics</b></summary>

- **XP retention on death:** Keep a percentage of your experience after dying. *(Toggleable)*
  - **Configurable retention percentage:** Change how much XP is kept ranging from 0% to 100%.

</details>

<details>
<summary>🧪 <b>Bottles o' Enchanting</b></summary>

- **Fixed XP from bottles:** Bottles o' Enchanting grant a consistent amount of experience instead of a random value from a range. *(Toggleable)*
  - **Configurable bottle XP:** Set the exact amount of XP each bottle grants.

- **Optional removal from Villagers:** Removes Bottles o' Enchanting from Cleric Villager trade pools. *(Toggleable)*

- **Brewable Bottles o' Enchanting:** Brew Bottles o' Enchanting by using an Echo Shard on any Awkward Potion variant!

<details>
<summary>📚 <b>Enchanting & Books</b></summary>

- **Additive book merging:** Combine enchanted books of different levels additively (e.g. I + II = III). *(Toggleable)*

- **Smart Mending:** Mending prioritizes repairing the most damaged item instead of selecting randomly. *(Toggleable)*

</details>

<details>
<summary>🏹 <b>Infinity Enchantment Improvements</b></summary>

- **No arrows required:** Bows or crossbows with Infinity can fire even with no arrows in inventory. *(Toggleable)*

- **Any arrow type:** Infinity can work with any arrow variant. *(Toggleable)*

- **Crossbow support:** Infinity can be applied to crossbows. *(Toggleable)*

- **Infinity + Mending compatibility:** Removes the vanilla mutual exclusivity between the two enchantments. *(Toggleable)*

</details>

<details>
<summary>⚒️ <b>Anvil Improvements</b></summary>

- **Remove repair penalty:** Eliminates the increasing XP cost for repeatedly repairing items. *(Toggleable)*

- **Cheap renaming:** Renaming items without other modifications costs no XP. *(Toggleable)*

- **No "Too Expensive!" limit:** Removes the vanilla anvil cost cap. *(Toggleable)*

</details>

<details>
<summary>🔥 <b>Furnace Quality of Life</b></summary>

- **Stored XP drops from furnaces:** Breaking a furnace releases accumulated smelting XP. *(Toggleable)*

</details>

<details>
<summary>⭐ <b>XP Collection</b></summary>

- **Experience magnet:** Increases the range at which XP orbs are pulled toward the player.
  - **Configurable magnet range:** Adjust the attraction range for XP orbs.

</details>

<details>
<summary>🌾 <b>Farming Rewards</b></summary>

- **Crops grant XP:** Harvesting fully grown crops can drop experience. *(Toggleable)*

- **Configurable crop XP:** Adjust how much XP crops provide.

</details>

## ⚙️ Configuration

All features can be configured through the mod's config file.

You can:
- Enable or disable individual features
- Adjust XP values
- Tune gameplay balance for modpacks or servers

## 🧩 Compatibility

The mod is designed to stay close to vanilla mechanics and should work with most mods. Although, mods that change the behaviour of:
- Experience mechanics
- Enchanting systems
- Anvils
- Villager trades

may override or conflict with some features.

## 🚧 Planned Features

Features currently being considered or planned for future updates:

- **Toggleable Enchanted Golden Apple recipe:** Allow enabling or disabling the crafting recipe through the config.
- Additional configuration options for existing mechanics.
- Further small quality-of-life tweaks to XP and enchanting systems.
